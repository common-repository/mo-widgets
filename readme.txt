==Description ==
Useing it can easier to control your Widgets.Adds checkboxes to choose your Widget to show or hide on different pages;and add a Widget with support for including PHP.
_Warning: PHP Widget provides great flexibility in customizing your sidebar.
It achieves this by automatically evaluating any included PHP included in the title or the body of the widget.
Care should be taken when including PHP code in the sidebar, as with anywhere else in the Wordpress system.
You may enter PHP code both in the title and the body.　　　　　　　
插件说明:
灵动边栏插件,使用它将使你更容易的控制你的边栏显示,安装激活以后会在小工具页面显示控制各个小工具在博客中任何页面是否显示,以实现不同页面不同侧边栏的效果;
同时在小工具页面产生一个支持php语言的文本小工具,让你更容易的添加属于自己的边栏小工具.
此插件在Display widgets和Samsarin PHP Widget基础上修改,在此感谢原作者.
注意:自定边栏小工具的标题栏也是支持php语言的;自定边栏小工具允许php代码在侧边栏执行,所以请小心使用.
== Installation ==
1. Download the  MO-widgets  and unzip it.
2. Copy file `MO-widgets` to the `/wp-content/plugins` directory.It default language is Chinese,if you want to let it show English,please delete or rename`MO-widgets-cn.php`and  rename `MO-widgets-EN.php.txt` to `MO-widgets-EN.php`.
3. Login to your admin console, go to `Plugins`, and enable the `MO-widgets`.
　　◎插件安装◎:　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　
①.下载 MO-widgets 插件,并解压.　　　　　　　　　　　　　　　　　　　　　
②.将解压到的文件夹MO-widgets上传到/wp-content/plugins 目录中,默认显示中文,如果你想要显示英文,请删除或者重命名MO-widgets-cn.php然后重命名MO-widgets-EN.php.txt为MO-widgets-EN.php.　　　　　　　　　　　　　　　　　　　
③.登陆后台,激活 MO-widgets插件.
